# Bifrost Enterprise

This section indexes the enterprise capabilities added on top of the
open-source Bifrost gateway. The full design lives in the spec at
[`specs/001-enterprise-parity/`](https://github.com/maximhq/bifrost/tree/main/specs/001-enterprise-parity).

## Deployment Modes

Bifrost ships one binary that runs in three modes:

| Mode | Use case | Multi-org | License | Phone-home | Metering/Billing |
|------|----------|:---------:|:-------:|:----------:|:----------------:|
| `cloud` | Vendor-hosted SaaS | enabled | not required | enabled | enabled |
| `selfhosted` | Customer on-prem | disabled (single org, schema-ready) | required | off by default | disabled |
| `airgapped` | Customer on-prem, no outbound | disabled | required | forbidden | disabled |

See [deployment-modes](./deployment-modes.mdx) for full details.
